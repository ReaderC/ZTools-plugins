# 截图工具

一个用于 ZTools 的截图标注插件，支持截图、图片导入、标注编辑、复制和保存。

## 功能

- 截图并打开标注编辑器。
- 支持拖入图片或通过图片输入打开编辑器。
- 支持矩形、圆形、箭头、直线、画笔、马赛克、文字和序号标注。
- 标注支持二次选中、移动、缩放和删除。
- 文字和序号标注支持双击编辑内容。
- 支持撤销、复制并关闭、保存到本地。
- 支持浅色和深色模式下的磨砂玻璃工具栏。

## 指令

- `截图` / `截屏` / `sc` / `shortcutCapture`：开始截图并进入编辑器。
- `截图并复制` / `scc`：截图后直接复制。
- `截图并保存` / `scs`：截图后保存到本地。
- 图片输入：打开图片到截图工具。

## 使用说明

进入编辑器后，选择工具栏中的标注工具进行绘制。绘制完成后不会自动选中标注，需要点击已有标注才会选中。

选中标注后可以拖动移动位置，拖动四角控制点调整大小。直线和箭头支持拖动端点调整方向和长度。按 `Delete` 或 `Backspace` 删除当前选中的标注。

文字和序号标注双击后可以修改内容，回车或失焦提交，`Esc` 取消编辑。

## 开发

```bash
npm install
npm run dev
```

开发模式入口在 `public/plugin.json` 的 `development.main` 中配置。

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`，包含 `plugin.json`、`preload.js`、`logo.png`、前端资源和 `README.md`。
