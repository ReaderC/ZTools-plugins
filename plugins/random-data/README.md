# 首次使用前，请先打开一次 `随机数据设置` 页面，完成指令注册

# 随机数据

用于快速生成常见测试数据，并自动填充到当前输入框。

## 怎么用

在 ZTools 中输入对应指令，就会直接生成一条随机数据。

可用于：

- 姓名、手机号、邮箱、用户名、密码
- UUID、时间戳、日期、URL、IP、MAC
- 地址、身份证号、银行卡号、公司名、职位、车牌号
- 短句、段落、JSON、CSV、Token、Base64 等

## 设置入口

输入下面任意指令可打开设置页：

- `随机数据设置`
- `数据指令设置`
- `rdsetting`

## 设置页可以做什么

- 开启或关闭某个数据项
- 自定义每个数据项的触发指令
- 点击“生成数据”先预览结果

所有项目默认开启。
